## üé® Color/Variant Guidelines

Many icons should have multiple color variants based on context:

**Standard Variants:**
- **Default**: Bronze (#C18F59) and Sand (#D2B48C)
- **Active/Hover**: Brighter bronze with glow
- **Disabled**: Desaturated gray
- **Danger**: Blood Red (#8E1C1C)
- **Success**: Olive Green (for health, success states)

**Rarity Variants** (for equipment-related icons):
- Common: Gray/white
- Uncommon: Green
- Rare: Blue
- Epic: Purple
- Legendary: Gold with sparkle effects

## üîß Technical Specifications

**File Format**: PNG with transparency
**Size Standards**:
- Icons: 16x16, 24x24, 32x32 base sizes
- Buttons/Panels: 9-slice compatible or multiple sizes
- Effects: Sprite sheets for animations

**Naming Convention**:
- `[category]-[name]-[size].png` (e.g., `currency-gold-coin-24.png`)
- `[category]-[name]-[variant]-[size].png` (e.g., `weapon-sword-icon-rare-32.png`)
- Animated: `[category]-[name]-sheet-[frames]x[size].png`

**Palette Adherence**: All sprites should use or complement the Blood & Bronze palette:
- Background: #121212, #1E1B18
- Accents: #8E1C1C, #C18F59, #D2B48C

## üìù Notes

- All sprites should feel hand-crafted with slight imperfections (not perfectly geometric)
- Maintain consistent light source (top-left) for depth
- Use dithering sparingly for gradients (16-bit aesthetic)
- Keep animations snappy (120-200ms total duration)
- Ensure all icons are readable at 16x16 minimum size
- Consider accessibility: don't rely on color alone for critical information
